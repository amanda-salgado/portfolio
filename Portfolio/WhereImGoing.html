Where I'm Going

I plan to continue learning to code and practicing. I am starting to feel confident with HTML and CSS, so my first goal is to master those. After that, I would like to start learning more JavaScript, and then figure out what other languages I should and can learn.

Girls Who Code

I learned about a program to teach girls coding about a decade ago, and always thought it would be fun to bring to the schools that I have worked at, but I have never had the confidence to be able to start helping others learn coding because I didn't know anything about it. While I'm still super new and have so many things to learn, I think being in Tech-Moms has given me the confidence to try it, and I'm considering starting a Girls Who Code club for female students at my school next year.
